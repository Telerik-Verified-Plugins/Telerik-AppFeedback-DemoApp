Telerik AppFeedback Plugin Demo

The AppFeedback plugin allows you to have your users send you feedback right from your app. Users can annotate screenshots and leave comments which show up 1:1 in your Telerik backend.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android - App package.

Initialize

Normally you'd want to do this on deviceready (see the doc), but for clarity we've created a dedicated button here.

Send feedback

Instead of pressing this button you can configure the plugin to enable shaking the device to send feedback.
